## Connect to Synology NAS with windows in few steps.
1. Intsall RaiDrive
  - https://www.raidrive.com/download

2. Click Add 

   ![00 Add](00.png)

3. Choose NAS

   ![01 NAS](01.png)

4. Input `Address`:`ip`, `port`, `path`. Account info: `username`, `password`

   ![02 Type Info](02.png)

5. Connect!

6. Check `local disk`

   ![03 Done!!](03.png)
